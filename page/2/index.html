Java NIO-4 Scatter / Gather

Posted on 2018-04-11

Java NIO 具有内置的scatter / gather支持。scatter / gather的概念用于从channel中读取和写入数据。

从channel的一个scatter读取操作指数据同时被读取到多个Buffer中。因此，channel将数据分散（scatter）到多个Buffer中。

从channel的一个gather操作指多个Buffer中的数据被写入到一个channel中。因此，channel从多个Buffer中收集（gather）数据。

scatter / gather在你需要分别处理传输的数据时会非常有用。例如，如果一条消息有头部和正文组成，你需要将头部和正文在独立的Buffer中。这样你可以很容易的单独操作头部和正文。
