Java NIO-14-Path

Posted on 2018-04-11 |

Java Path 接口是在 Java 6 和 Java 7 中的部分更新之一。Java Path 接口是在 Java 7 中添加到 Java NIO 中的。Path 接口在java.nio.file包下，所以它的完整包名是java.nio.file.Path。

一个 Java Path 实例代表文件系统中的一个路径（path）。一个路径可以指向一个文件或者一个文件夹。一个路劲更可以是绝对的或者相对的。绝对路径包含了从文件系统的根目录到它指向的文件或路径的完整的路径。相对路径包含了指向的文件或路径相对于其它路径的路径。相对路劲可能听起来有一些疑惑。但不用担心，我会在这篇教程中进行更详细的阐释。

不要混淆文件系统的路径和某些系统的环境变量。java.nio.file.Path接口和环境变量没有任何的关系。

在很多方面java.nio.file.Path接口和java.io.File很相似，但是仍有一些不同的地方。尽管如此，在很多情况下你可以把 File 类的使用替换为 Path 接口。
