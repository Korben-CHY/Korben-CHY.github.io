Timeout for tests

我们还可以通过JUnit测试函数的运行时间，如果一个函数的运行时间太长，JUnit会报失败。

@Test(timeout = 1000)

我们可以在@Test注解中指定函数运行的最大时间，以毫秒为单位：

Timeout Rule

我们同样可以使用Timeout Rule来测试函数的运行时间。与timeout不同的是，Timeout Rule会作用于当前测试类的每一个方法。而且Timeout Rule会覆盖timeout

注意：如果@Test(timeout = xx)与Timeout Rule同时出现，则高优先级的Timeout Rulle起作用。

Timeout Rule作用于当前类的所有方法，包括@Before和@After。
